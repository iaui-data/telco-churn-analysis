# 📊 Telco 客户流失归因分析与高风险预警 (SaaS Churn Analysis)

## 📌 项目背景
基于 7,043 条电信/SaaS 用户订阅业务数据，从 0 到 1 构建流失分析体系。独立完成特征工程、异动归因诊断与高危用户圈选，最终输出包含动态交互与精准策略落地的商业决策看板 (BI Dashboard)。

## 💡 核心业务洞察
- **月付模式为流失重灾区：** 月付合约用户流失率高达 42.71%，是两年合约（2.85%）的近 15 倍。
- **增值服务具备显著留存效应：** 无技术支持的月付用户流失率攀升至 50.37%，开通后显著压降至 30.70%。
- **核心流失窗口期：** Tableau 散点图显示，流失客户高度集中于【在网时长 < 12 个月】且【月消费 > 60 元】的区间。

## 🛠️ 技术栈与核心能力
- **MySQL:** CTE 逻辑嵌套, 窗口函数 (`NTILE`), `CASE WHEN` 复杂特征工程
- **Tableau:** 动态参数联动, 散点分布识别, 交互式决策看板设计
- **业务思维:** 异动归因分析, ROI 驱动的精细化用户分层策略

## 📂 仓库目录说明

- 📁 **`sql/results/`** — SQL 跑出的核心结论表与业务解读
